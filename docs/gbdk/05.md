# ゲームボーイで動くプログラムを書いてみる

## 05. 実践1の解答例

解答例:

```c
#include <gb/drawing.h>

int main() {

    gotogxy(3,3);
    color(3,0);
    gprintf("Tokyo");

    gotogxy(5,5);
    color(3,1);
    gprintf("City");

    gotogxy(7,7);
    color(0,3);
    gprintf("University");
}
```

特に特筆するところはないですが、大事なのは**gotogxyの座標指定は0から始まる**ということです。
